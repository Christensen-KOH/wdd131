Site Plan: Speak up with Chris

Site Name

Speak up with Chris

Description: This name centers the relationship between the tutor and the student. It is approachable, encourages active participation ("Speak up"), and clearly identifies the personalized nature of the English tutoring services provided. It is memorable and professional.

Optional domain availability: speakupwithchris.com

Site Purpose

The site serves as a comprehensive, dynamic portal for English language learners and their instructor. It provides a seamless student experience for booking interactive, conversational English lessons, accessing custom-tailored homework and grammar resources, and tracking learning progress. For the teacher, it acts as a backend CRM and scheduling tool to manage student data, availability, and session notes.

Scenarios

These questions represent the target audience (adult English learners, specifically Russian-speaking professionals) seeking to use the platform:

  • Scenario 1: "How can I easily see Chris's available time slots this week and book my next conversational English lesson?"
  • Scenario 2: "Where can I log in to download the engineering vocabulary worksheet from our previous lesson and upload my completed homework?"
  • Scenario 3: "How do I check my lesson history to see how many classes I have completed this month?"

Color Schema

The color palette is designed to build trust while maintaining a welcoming, educational environment.

  • Primary Color (Teal/Turquoise - #008080): Used for call-to-action buttons (e.g., "Book a Lesson"), active navigation links, and section highlights. It represents clarity and communication.
  • Secondary Color (Slate Gray/Navy - #2F4F4F): Used for the main header, footer background, and primary headings (H1, H2). It grounds the design and represents professionalism and structure.
  • Background Color (Off-White/Light Gray - #F8F9FA): Used for the overarching body background to keep the interface clean and reduce eye strain when reading text.

Typography

The typography pairs a bold, modern font for structure with a highly readable font for learning materials.

  • Heading Font: Montserrat. Used for the site logo text, primary headings (H1, H2, H3), and navigation menu items. It provides a clean, structural, and modern look.
  • Body Font: Open Sans. Used for all paragraphs, lists, form labels, and instructional text. It is a highly legible sans-serif font, which is crucial for users who are reading in a second language.

Wireframes (Home/Booking Page Layout)

Below are structural representations of how the main student dashboard will adapt across devices.

Desktop View

  • Header: Logo on far left. Navigation links (Dashboard, Book Lesson, Resources, Logout) aligned to the right.
  • Hero Section: Welcome message ("Welcome back, [Student Name]") spanning the top.
  • Main Content (Two Columns):
    • Left Column (60%): Interactive calendar displaying available dates and time slots.
    • Right Column (40%): "Next Lesson" countdown card and a list of recent resource downloads.
  • Footer: Centered copyright info and contact links.

Mobile View

  • Header: Logo centered. Hamburger menu icon on the top right for navigation.
  • Hero Section: Stacked welcome message.
  • Main Content (Single Column):
    • Top Block: "Next Lesson" summary card for immediate visibility.
    • Middle Block: A simplified, vertically scrolling list of available dates/times instead of a full grid calendar.
    • Bottom Block: Recent resource downloads.
  • Footer: Stacked copyright and contact links.
